Communication between Components

I started learning Angular this year. When I have a question, I always go to angular's official site to find the answer, and I believe that it is the best source about Angular. However, I had trouble finding simple methods to make components communicate with each other, because angular's official site includes few methods and doesnt explains which method is suitable for which use case. This is the official documentation about Component Interaction.

Now, I am more experienced so I made this tutorial to help novice web developers choose a method depending on their needs. It includes 6 different methods from easiest to hardest.

You can navigate through the pages with left and right arrows (← and →).
